Объекты в S3

Список — GET /api/objects; загрузка с диска — POST /api/upload-local; копирование по префиксу — POST /api/copy-prefix

Загрузка с диска

Положите файлы .jpg, .jpeg, .png или .gif в каталог assets/source/ (или путь из LOCAL_SOURCE_DIR). Сервер перекодирует их в JPEG (качество 75) и загрузит под префикс uploads/.

    Копирование по префиксу

    Тело запроса JSON: {"from":"uploads/","to":"archive/"}. Опционально "delete_source": true — удалить исходные объекты после успешного копирования.
